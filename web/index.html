ORYN

Non-custodial autonomous treasury agent on Celo. Every deposit into a vault is split automatically, onchain, per rules only the vault owner controls.

Deposit into a vault

Pay into an ORYN vault (approve + deposit). ORYN's agent detects it and executes the split within ~15s.

Set vault rules

Vault owners only — describe your split in plain English (e.g. "pay designer 30%, rest stays as savings"), review the parsed table, then sign from your wallet.
